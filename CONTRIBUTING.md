# Ways to Contribute

## 💡 Feature Ideas

Have an interesting idea?

Open an Issue and let's discuss it.

Some of the best features start as simple conversations.

---

## 🐞 Report Bugs

Found something broken?

Open a Bug Report with:

- Steps to reproduce
- Expected behaviour
- Screenshots (if applicable)

---

## 🔧 Submit a Pull Request

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Test everything.
5. Open a Pull Request.

For UI changes, screenshots or a short screen recording are always appreciated.

# Code Guidelines

- Write clean, readable TypeScript.
- Keep components reusable.
- Follow the existing project structure.
- Test before submitting.
- Be respectful during discussions.
