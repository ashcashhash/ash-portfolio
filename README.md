# Ashnish Kumar Sharma — Personal Portfolio

Welcome to the source code of my personal portfolio!

This project is more than a portfolio—it's a playground where software engineering meets music. I wanted to build an experience that reflects both my professional journey as a Full-Stack Engineer and my passion for music.

🌐 **Live Website:** https://www.ashcashhash.com

## ✨ Features

- 🎸 Interactive playable guitar
- ⚡ Smooth page transitions & micro-interactions
- 🌙 Modern UI
- 📱 Fully responsive design
- 🎨 Custom animations powered by Motion
- 🚀 Fast performance with Vite
- 💼 Experience, Projects & Certifications
- 📬 Contact section

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Lucide Icons

---

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/ashcashhash/ash-portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
