* 🎸 Interactive playable guitar
* ⚡ Smooth page transitions & micro-interactions

* 📱 Fully responsive design
* 🎨 Custom animations powered by Motion
* 🚀 Fast performance with Vite
* 💼 Experience, Projects & Certifications
* 📬 Contact section

⸻

🛠 Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Motion
* Lucide Icons

⸻

🚀 Running Locally

Clone the repository:

git clone https://github.com/ashcashhash/<repo-name>.git

Install dependencies:

npm install

Start the development server:

npm run dev

Build for production:

npm run build
